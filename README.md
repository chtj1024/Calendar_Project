#웹 및 데스크탑 캘린더 구현 프로젝트
## 프로젝트 소개
데스크탑에서 활용하는 캘린더가 마땅히 없다고 생각하여 react와 electron을 활용해 웹, 데스크탑(윈도우)환경에서 캘린더의 일정을 등록할 수 있는 서비스를 구현하였다.

## 담당 (개인)
- 모달창 구현 (이벤트를 추가, 수정, 삭제하기 위한)
  - 캘린더 클릭 및 드래그 시 해당 날짜의 이벤트 제목, 내용 등록 구현
  - 이벤트 클릭 시 제목, 내용 확인 및 수정, 삭제 구현
  - 드래그 하지 않아도 이벤트 모달창에서 날짜 변경 기능 구현
- 윈도우 위젯 구현 (Electron)
  - Tray 기능 구현 (윈도우 우측 하단 숨겨진 아이콘에 표시되는 기능)
    - 커서 올릴 때 앱 제목 표시
    - Tray 우클릭 시 메뉴 표시 및 메뉴로 최소화 된 창 열기, 종료 기능 구현
    - 캘린더 앱이 실행됐을 때 작업표시줄엔 표시하지 않고 Tray에만 표시되고, Tray 클릭해서 실행했을 때는 작업표시줄에 보이게 설정
  - 윈도우바 제거 및 설정 버튼 구현
    - 크기 및 위치 설정 기능 구현
    - 최소화, 종료 버튼 구현
    - 투명도 조절 기능 구현
    - 캘린더의 현재 위치, 사이즈, 투명도를 기억해서 재시작해도 환경 동일할 수 있게 구현
- DB 연동
  - Mysql, SQLite를 활용하여 CRUD 구현과 사용자 PC에 일정 이벤트가 저장될 수 있게 구현

## 설치 시 실행방법
node_modules 폴더가 없어 새로 설치 시 'npm install' 명령어를 통해 node_moduels 폴더를 추가하면 된다.
