#웹 및 데스크탑 캘린더 구현 프로젝트
## 프로젝트 소개
데스크탑에서 활용하는 캘린더가 마땅히 없다고 생각하여 react와 electron을 활용해 구현하였다.

## 담당
- 모달창 구현 (이벤트를 추가, 수정, 삭제하기 위한)
  - 이벤트의 제목, 내용 설정 가능
  - 이벤트의 날짜 설정, 수정 가능
- 윈도우 위젯 구현 (Electron)
  - Tray 기능 구현 (윈도우 우측 하단 숨겨진 아이콘에 표시되는 기능)
    - 커서 올릴 때 앱 제목 표시
    - Tray 우클릭 시 메뉴 표시 및 메뉴로 실행, 종료 가능
    - 캘린더 앱이 실행됐을 때 작업표시줄엔 표시하지 않고 Tray에만 표시되고, Tray 클릭해서 실행했을 때는 작업표시줄에 보이게 설정
- 윈도우바 제거 및
